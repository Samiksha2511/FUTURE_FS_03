# FUTURE_FS_03 – Local Business Connect

## 📌 Project Overview

Local Business Connect is a professional web application developed as part of the **Future Interns Full Stack Web Development Internship – Task 3**. The website helps customers discover trusted local businesses and enables business owners to promote their services online.

## ✨ Features

- Responsive and modern design
- Home, About, Services, Businesses, Register, and Contact pages
- Business search functionality
- Featured local businesses
- Business registration form
- Clean and user-friendly interface
- Node.js and Express.js backend

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- JSON

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/Samiksha2511/FUTURE_FS_03.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
npm start
```

4. Open your browser and visit

```
http://localhost:3000
```
